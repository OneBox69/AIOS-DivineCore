## What goes here

- `discord-middleware/index.js` (and any helpers) — mirror of VPS code
- nginx site configs (sanitised — no real cert paths or secrets)
- PM2 ecosystem files
- Any deploy scripts (rsync, docker pull + swap, etc.)

## What does NOT go here

- The v2 Python runtime — that's in `divinecore-v2/`.
- Secrets, API keys, certs, or `.env` files. Reference only.
- VPS provider account details beyond what's already in CLAUDE.md §10.
