# shared/context/playbooks — Channel Playbooks

Channel-specific tactical playbooks for content and outreach. Each file extends [`../identity/voice.md`](../identity/voice.md) + the relevant persona ([`../identity/pang.md`](../identity/pang.md) or [`../identity/mayank.md`](../identity/mayank.md)) with platform-specific structure.

## Files

| File | What it contains |
|------|------------------|
| `linkedin-playbook.md` | Pang's LinkedIn tactical playbook — hooks / CTAs / closings swipe sections, 3 post frameworks (PAS, SLA, Case Study), writing flow, profile copy, growth strategy, differentiation requirement, pre-publish checklist. |
| `x-playbook.md` | Pang's X (Twitter) tactical playbook — flywheel (content→profile→email→offer→proof), breakpoint diagnosis, X writing guidelines, naughty word list. v0 strategic frame; swipe files come later. |
| `upwork-loom-script.md` | The standardized Upwork Loom video script — ~5:30 length, eight sections (open, JD-specific demo, offer+guarantee, contrast, metaphor, RM AIOS three-layer demo, tie-back, close). Variables to fill per recording. |
| `swipe-file.md` | Cross-channel raw content examples — LinkedIn / X / YouTube / cross-channel patterns. Holds the *moves* (real posts/threads with notes on what specifically works). Templates stay in per-channel playbooks. |

## Loading discipline by task

| Task | Files to load |
|------|---------------|
| Pang LinkedIn post | `../identity/pang.md` + `linkedin-playbook.md` + `../identity/voice.md` + (optional) `swipe-file.md` §01 |
| Pang X post | `../identity/pang.md` + `x-playbook.md` + `../identity/voice.md` + (optional) `swipe-file.md` §02 |
| Mayank YouTube content | `../identity/mayank.md` + `../identity/voice.md` + `../identity/strategy.md` + (optional) `swipe-file.md` §03 |
| Upwork Loom recording | `upwork-loom-script.md` + `../sales-and-delivery/guarantee.md` |

**Pattern: persona file always first**, then the channel playbook, then optional swipe examples. Persona owns *who is writing*; playbook owns *how the channel demands the writing be structured*.
