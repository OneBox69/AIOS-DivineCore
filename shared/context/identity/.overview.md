# shared/context/identity — Brand & Persona

Who DivineSide is, how we sound, who we sell to, plus per-person personas for Mayank and Pang. Every agent generating human-facing copy starts here.

## Files

| File | What it contains |
|------|------------------|
| `business-info.md` | What DivineSide is, what we sell vs. don't, the AI OS positioning, the agency-vs-AIOS distinction |
| `voice.md` | Brand-level tone, banned phrases, per-channel formatting rules. The single source of truth for *how* anything sounds. |
| `audience.md` | Who we sell to — sub-segments (agency owner, eCom founder, solo operator), pain points, what attracts vs repels them |
| `strategy.md` | Current positioning, distribution channels, content cadence, monetisation layers |
| `mayank.md` | Mayank's persona — channel-agnostic worldview, signature lines, role, POV. Load for any Mayank-voiced content. |
| `pang.md` | Pang's persona — channel-agnostic personality (5 expanded traits), writing style, content pillars, hot takes, constant reinforcement rule. Load for any Pang-voiced content. |

## Loading discipline

- **Any writing agent loads `voice.md`** (brand tone). No exceptions.
- **First-person content loads the relevant persona** — `mayank.md` or `pang.md` — for whoever's voicing it.
- **Audience-aware copy** (cold email, sales pitches) loads `audience.md`.
- **Public/external framing** leads with the outcome ("your business, running on AI"). See [../sales-and-delivery/offer.md](../sales-and-delivery/offer.md) §01 for the naming discipline rule.

## What does NOT go here

- Channel-specific tactical playbooks (hook patterns, CTAs, post frameworks) → [`../playbooks/`](../playbooks/)
- Pricing, offers, guarantee terms, sales scripts → [`../sales-and-delivery/`](../sales-and-delivery/)
- Architecture / system specs → [`../../../CLAUDE.md`](../../../CLAUDE.md)
- Knowledge bases (Kallaway, Hormozi frameworks) → module folders (`branding_os/knowledge-base/`, `sales_os/knowledge-base/`)
- Working examples / past content → Supabase + module folders
